My hobbies mainly revolve around computers. I love both building them and using them for programming and others tasks. I enjoy all types of programing, but I tend to enjoy more algorithmic programming and some game development. I currently have one past attempt at making a game on my GitHub and I am gearing up towards a new game idea as I really want to get a more finished product.
