About me

My name is Oliver Odendaal and I am a current student at University of Waterloo taking Computer Science (2025). I have mainly done algorithmic programming, but have done a wide variety of other topics. I enjoy taking on different projects whether I get to finish them or not. I currently know how to program well in Python, Java and C++ and have been learning HTML, Javascript and CSS. I have been trying to transition to Linux instead of Windows as it gives me a lot more freedom (and I enjoy the learning curve).

Hobbies/Interests

My hobbies mainly revolve around computers. I love both building them and using them for programming and others tasks. I enjoy all types of programing, but I tend to enjoy more algorithmic programming and some game development. I currently have one past attempt at making a game on my GitHub and I am gearing up towards a new game idea as I really want to get a more finished product. I also enjoy rock climbing, bouldering, scuba diving and motorcycling (both on-road and off-road).
